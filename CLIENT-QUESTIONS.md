## Donate flow

The "Make a Donation" button can be setup in 2 ways:

**Option A — Accept donations directly on the site.**
We can wire the button to a working donation flow (Squarespace has a built-indonation block).**merchant services** would be set up first (bank/processor account, fee structure, receipt
templates, 501(c)(3) confirmation for the donor's tax purposes).

**Option B — Redirect to an intake form.**
The Donate button can point to a short intake form ("Tell us a bit about how you'd like to give"). Submissions lands in the foundation's inbox and someone calls the donor back to arrange the gift directly. Personal touch.

**Option C — Mailing address only.**
The button links to a page that says "We accept checks at this address." Simplest of all; appropriate if 95% of giving is going to come from people who already know the family.

Please indicate which option to develop. We can also stage Option B now and upgrade to Option A once merchant services are in place.
